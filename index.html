Guía de Etiquetas HTML

<html>

La etiqueta <html> define el documento como una página HTML. Debe contener todos los elementos de la página.

<head>

La etiqueta <head> contiene meta información sobre el documento, como el título, las hojas de estilo y las configuraciones de codificación.

<title>

La etiqueta <title> define el título del documento que aparece en la pestaña del navegador.

<meta>

La etiqueta <meta> proporciona metadatos sobre el documento, como la codificación de caracteres y la descripción.

<body>

La etiqueta <body> contiene el contenido visible de la página web, como texto, imágenes y enlaces.

<h1> - <h6>

Las etiquetas <h1> a <h6> definen encabezados, siendo <h1> el más importante y <h6> el menos importante.

<p>

La etiqueta <p> define un párrafo de texto.

<a>

La etiqueta <a> define un enlace a otra página o recurso. Se usa con el atributo href.

Ejemplo: <a href="https://www.ejemplo.com">Enlace</a>

<img>

La etiqueta <img> inserta una imagen en la página. Se usa con los atributos src y alt.

Ejemplo: <img src="imagen.jpg" alt="Descripción de la imagen">

<ul> y <li>

La etiqueta <ul> define una lista desordenada y <li> define un elemento de la lista.

Ejemplo:

<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ul>

<ol> y <li>

La etiqueta <ol> define una lista ordenada y <li> define un elemento de la lista.

Ejemplo:

<ol>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ol>

<div>

La etiqueta <div> define un contenedor genérico que agrupa contenido y se utiliza para aplicar estilos o scripts específicos.

<span>

La etiqueta <span> es un contenedor en línea usado para agrupar texto y aplicar estilos.

<form>

La etiqueta <form> define un formulario HTML para la entrada de datos del usuario.

<input>

La etiqueta <input> define un campo de entrada de datos en un formulario.

Ejemplo: <input type="text" name="nombre">

<button>

La etiqueta <button> define un botón que se puede hacer clic.

Ejemplo: <button>Clic aquí</button>

<table>, <tr>, <td>

La etiqueta <table> define una tabla. <tr> define una fila de la tabla, y <td> define una celda de la tabla.

Ejemplo:

<table>
  <tr>
    <td>Celda 1</td>
    <td>Celda 2</td>
  </tr>
</table>

<footer>

La etiqueta <footer> define el pie de página del documento o una sección.

<header>

La etiqueta <header> define el encabezado de un documento o una sección.

<nav>

La etiqueta <nav> define una sección de navegación que contiene enlaces a otras partes del sitio web.

<section>

La etiqueta <section> define una sección del documento.

<article>

La etiqueta <article> define contenido independiente, como artículos o publicaciones de blog.

<aside>

La etiqueta <aside> define contenido que está relacionado con el contenido principal pero que es secundario, como una barra lateral.
